========================
Which zip file choose
========================

The number in the zipped file name is the sequential build number. The higher means the more upgraded version.
There is not distintion between stable and unstable versions. All the ones published are quite stable and tested.

The file with "source" in the name contains the source code to be compiled.

The file with "exec" in the name contains the executable code to run.

========================
How to run ICE
========================

Download the Exec file. Uncompress in a folder.

Run with the following command:

java -jar "ICE.jar"
